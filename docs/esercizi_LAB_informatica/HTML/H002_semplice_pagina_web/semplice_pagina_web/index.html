Lezione 1

HTML

HTML è un linguaggio di Markup che permette di descrivere un documento e come il contenuto del documento è organizzato (titolo, sottotitolo, paragrafo, ...).

Gli elementi che compongono un documento HTML sono chiamati tag e vengono principalmente utilizzati per definire la struttura del documento.

Per definire lo stile delle varie parti del documento si usano i fogli di stile CSS che contengono regole. Ogni regola indica gli stili da applicare agli elementi che vengono identificati mediante un selettore.
